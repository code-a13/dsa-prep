# Arrays

---

## 🔹 Problem 1: Sum of Array Elements

**Problem:**  
Given an array, find the sum of all elements.

**Example:**

Input: [1, 2, 3, 4, 5]  
Output: 15  

---

## 🔹 Problem 2: Count Odd Numbers

**Problem:**  
Count how many odd numbers are present in the array.

**Example:**

Input: [1, 2, 3, 4, 5]  
Output: 3  

---

## 🔹 Problem 3: Check if Array is Sorted

**Problem:**  
Check whether the array is sorted in ascending order.

**Example 1:**

Input: [1, 2, 3, 4, 5]  
Output: True  

**Example 2:**

Input: [5, 4, 6, 7]  
Output: False  


---

## 🔹 Problem 4: Reverse an Array

**Problem:**  
Reverse the array in-place.

**Example:**

Input: [1, 2, 3, 4, 5]  
Output: [5, 4, 3, 2, 1]
